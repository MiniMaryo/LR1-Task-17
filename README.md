# Задание 17 
Составить консольное приложение, которое содержит описание класса Car (автомобиль),
который должен содержать:
•	Открытые поля для хранения марки, гос.номер и пробега
•	Метод Show() – показать на экране данные об автомобиле
•	Метод расчета среднего пробега для заданного возраста автомобиля в качестве параметра
Что сделать:
1.	В методе Main() нужно создать объект класса Car.
2.	Создать объект класса Car.
3.	Задать свойствам объекта следующие значения: Марка = Жигули, Номер = Т144ТО, Пробег = 10350.
4.	Вывести на экран значения свойств объекта.
5.	Метод расчета средне годового пробега автомобиля, если его возраст 2 года и вывести на печать.
